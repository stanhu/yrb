#yawareness_update_with_clients(clients) ⇒ Object

Returns a serializable update object which is representation of a current Awareness state. Unlike [Y::Awareness#update], this method variant allows to prepare update only for a subset of known clients. These clients must all be known to a current Awareness instance, otherwise an error will be returned.
